Last updated on May 27, 2020

Below is a list of all the different subject areas in Scopus. You can search using field names and codes, or filter and browse by subject area.

Please note: To search using ASJC Codes, use SUBJTERMS ( ) in Scopus Advanced Search. For example, to search for articles under General Agricultural and Biological Sciences, enter SUBJTERMS(1100).
